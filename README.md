
# Olotap Demo

Olotap Tiptap HTML editor demo project.

# Installation

```sh
npm i
```

## Running development server

```sh
npm run dev
```

## Build for production

After the npm build command, all distribution files are created in the <b>/dist</b> folder.

```sh
npm run build
```
